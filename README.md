Development
-----------

Install [node.js](http://nodejs.org/download/)

Install dependencies:

    gem install compass
    npm install
    npm install -g bower
    bower install

Then, run `grunt build` in order to inject the bower dependencies into the proper blocks in `app/index.html`

`grunt serve` to run the dev server
